# Key interfaces from existing codebase
From elm/core/copy.py:
The three public functions have this return pattern:
- Success path: `return create_success_result(message, record_count=total_records)`
- Failure path: `return handle_exception(e, "... copy")`

From elm/core/history.py (created in Plan 01):
```python
class HistoryRecorder:
    def record(self, result: OperationResult, params: Dict[str, Any]) -> bool: ...
```

params mapping for HistoryRecord:
- operation_type: "db2db" | "db2file" | "file2db"
- source: params.get("source_env") or params.get("file_path") for file2db source
- target: params.get("target_env") or params.get("file_path") for db2file target
- query: params.get("query")
- table: params.get("table")
- mode: params.get("mode")
- batch_size: params.get("batch_size")
- parallel_workers: params.get("parallel_workers")
- start_time: ISO-8601 string captured at function entry
- end_time: ISO-8601 string captured just before return

Modify `elm/core/copy.py` to add history recording to all three public functions per D-15 and D-16. The recording must be non-blocking — any history-related exception is caught and the original result is still returned.

**Pattern for each function:**
1. At the top of the `try` block (first statement after `try:`), capture:
   ```python
   _history_start_time = datetime.now().isoformat()
   ```
2. At the end of the success path, BEFORE the `return` statement:
   - Create the `result = create_success_result(...)` as a local variable.
   - Add the recording block after creating `result` and before returning it.
3. In the `except Exception as e:` block, BEFORE the `return`:
   - Create `result = handle_exception(e, ...)` as a local variable.
   - Add the recording block after creating `result` and before returning it.

**Recording block for `copy_db_to_db`:**
```python
_history_end_time = datetime.now().isoformat()
try:
    recorder = HistoryRecorder()
    _history_params = {
        "operation_type": "db2db",
        "source_env": source_env,
        "target_env": target_env,
        "query": query,
        "table": table,
        "mode": mode,
        "batch_size": batch_size,
        "parallel_workers": parallel_workers,
        "start_time": _history_start_time,
        "end_time": _history_end_time,
    }
    result.history_saved = recorder.record(result, _history_params)
except Exception:
    result.history_saved = False
return result
```
