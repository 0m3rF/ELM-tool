2. **`read_records(self) -> List[HistoryRecord]`**:
   - Returns `self._read_records(self.config.get_history_file())`

3. **`get_record(self, record_id: int) -> Optional[HistoryRecord]`**:
   - Calls `read_records()`, loops to find matching `id`, returns the record or `None`

4. **`update_record(self, record_id: int, **kwargs) -> bool`**:
   - Acquire `file_lock(self.config.get_history_file())`
   - Read all records via `self._read_records(history_file)`
   - Find record with matching `id`
   - For each `key, value` in `kwargs`, if `hasattr(record, key)`, set it via `setattr`
   - Write back via `self._write_records(history_file, records)`
   - Return `True` on success, `False` if record not found or any exception occurs

All methods reuse existing `_read_records`, `_write_records`, and `file_lock`.

**Command 1: `list`**
```python
@copy.command(name="list")
@click.option("--status", help="Filter by status (success/failure)")
@click.option("--source", help="Filter by source environment or file")
@click.option("--target", help="Filter by target environment or file")
@click.option("--operation-type", help="Filter by operation type (db2db/db2file/file2db)")
@click.option("--table", help="Filter by table name")
@click.option("--limit", type=int, default=10, help="Max results (0 = unlimited)")
@click.option("--sort", type=click.Choice(["asc", "desc"]), default="desc", help="Sort order by date")
@click.option("--format", "output_format", type=click.Choice(["table", "json"]), default="table", help="Output format")
def list_history(status, source, target, operation_type, table, limit, sort, output_format):
    """List previous copy operations"""
```
Implementation logic:
1. Instantiate `HistoryRecorder()` and call `read_records()`
2. If no records, `click.echo("No history records found.")` and return
3. Apply filters: for each provided flag, check `getattr(record, field) == value` (case-insensitive for strings via `.lower()`). Skip `None` record fields.
4. Sort by `record.date` descending (default) or ascending
5. Apply limit if `limit > 0`
6. If `output_format == "json"`: `json.dumps([asdict(r) for r in records], indent=2)` and echo
7. If `output_format == "table"`: format as fixed-width columns. Columns: `ID`, `Date`, `Type`, `Source`, `Target`, `Table`, `Status`. Use `str.ljust`/`rjust` for alignment. Echo each row.

**Command 2: `re-run`**
```python
@copy.command(name="re-run")
@click.argument("id", type=int)
def re_run(id):
    """Re-run a previous copy operation"""
```
Implementation logic:
1. `recorder = HistoryRecorder(); record = recorder.get_record(id)`
2. If `record is None`: `click.echo(f"Error: History record {id} not found.", err=True)` and `raise click.ClickException("Record not found")` (non-zero exit)
3. Based on `record.operation_type`:
   - `"db2db"`: call `core_copy.copy_db_to_db(source_env=record.source, target_env=record.target, query=record.query, table=record.table, mode=record.mode or 'APPEND', batch_size=record.batch_size, parallel_workers=record.parallel_workers or 1)`
   - `"db2file"`: call `core_copy.copy_db_to_file(source_env=record.source, query=record.query, file_path=record.target, mode=record.mode or 'REPLACE', batch_size=record.batch_size, parallel_workers=record.parallel_workers or 1)` — Note: `file_format` defaults to `'csv'` since it is not stored in history.
   - `"file2db"`: call `core_copy.copy_file_to_db(file_path=record.source, target_env=record.target, table=record.table, mode=record.mode or 'APPEND', batch_size=record.batch_size, parallel_workers=record.parallel_workers or 1)` — `file_format` defaults to `'csv'`.
4. If `result.success`: update original record via `recorder.update_record(id, last_run_date=datetime.now().isoformat(), status="success", record_count=result.record_count, end_time=datetime.now().isoformat(), error_message=None)` and echo `result.message`
5. If `not result.success`: check if failure is due to missing environment/file (look for "not found" or "does not exist" in `result.message`). If yes, create a NEW failure record by calling `recorder.record(result, _history_params)` with the same params as the original but current timestamps (per D-07). Echo error. If failure is for another reason, echo error but do NOT create a new history entry.
6. For `db2db` and `file2db`, `apply_masks` defaults to `True`. For all commands, `verbose_batch_logs` defaults to `True`. Use default values for all unstored parameters (`validate_target=False`, `create_if_not_exists=False`, `encryption_key=None`).

**Command 3: `edit`**
```python
@copy.command(name="edit")
@click.argument("id", type=int)
@click.option("-s", "--source", help="Override source environment or file path")
@click.option("-t", "--target", help="Override target environment or file path")
@click.option("-q", "--query", help="Override SQL query")
@click.option("-T", "--table", help="Override target table name")
@click.option("-f", "--format", type=click.Choice(['CSV', 'JSON']), help="Override file format (for db2file/file2db)")
@click.option("-m", "--mode", type=click.Choice(['APPEND', 'OVERWRITE', 'FAIL']), help="Override write mode")
@click.option("-b", "--batch-size", type=int, help="Override batch size")
@click.option("-p", "--parallel", type=int, help="Override parallel workers")
@click.option("-k", "--encryption-key", help="Override encryption key")
@click.option("--no-mask", is_flag=True, help="Disable data masking")
@click.option("--validate-target", is_flag=True, help="Validate target table exists")
@click.option("--create-if-not-exists", is_flag=True, help="Create target table if missing")
def edit_history(id, source, target, query, table, format, mode, batch_size, parallel, encryption_key, no_mask, validate_target, create_if_not_exists):
    """Edit and re-run a previous copy operation"""
```
Implementation logic:
1. `recorder = HistoryRecorder(); record = recorder.get_record(id)`
2. If `record is None`: echo error to stderr and raise `click.ClickException`
3. Build parameter overrides: for each flag, if provided (not `None`), use it; otherwise use `record` value. For `no_mask`, if flag is `True`, set `apply_masks=False`; else default to `True`.
4. **Preview** (D-09): Echo a block showing the resolved parameters:
   ```
   Preview of edited operation:
   ─────────────────────────
   Operation type: {operation_type}
   Source: {source}
   Target: {target}
   Query: {query}
   Table: {table}
   Mode: {mode}
   Batch size: {batch_size}
   Parallel workers: {parallel}
   ─────────────────────────
   ```
5. Execute the same core function as `re-run` but using the overridden parameters.
6. On completion (success or failure), create a **NEW** history entry by calling `recorder.record(result, _history_params)` with current timestamps. Do NOT update the original record (per D-10).
7. Echo `result.message`. If success and `result.record_count`, echo `f"Records processed: {result.record_count}"`. If failure, raise `click.UsageError(result.message)`.
